Функціонал	                Опис

Додавання контакту	        Дозволяє користувачеві додати новий контакт з ім'ям та одним або декількома номерами телефону.
Редагування контакту	    Дозволяє користувачеві змінювати існуючий контакт, додаючи нові номери телефону або змінюючи існуючі.
Видалення контакту	        Користувач може видалити існуючий контакт з адресної книги.
Додавання дня народження	Дозволяє користувачеві додати день народження для контакту.
Показ дня народження	    Виводить день народження для вказаного контакту.
Пошук контакту за           Користувач може знайти контакт, використовуючи часткове співпадіння з ім'ям.
частковим ім'ям	
Сортування контактів	    Користувач може відсортувати контакти за ім'ям, номером телефону або днем народження.
Додавання групи	            Дозволяє користувачеві створити нову групу для організації контактів.
Видалення групи	            Користувач може видалити створену раніше групу.
Додавання контакту          Користувач може додати існуючий контакт до вже створеної групи.
до групи	
Видалення контакту          Користувач може видалити контакт з групи, до якої він належить.
з групи	
Повідомлення про наближені  Програма може сповіщати користувача про наближені дні народження контактів.
дні народження	
Збереження та завантаження  Можливість збереження та завантаження даних з файлу.
даних
